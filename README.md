# ぷよぷよ ブラウザゲーム

JavaScriptとCanvasでSEGAの「ぷよぷよ」ゲームを再現しました。

## 操作方法

| キー       | 操作       |
|------------|------------|
| →          | 右へ移動   |
| ←          | 左へ移動   |
| ↓          | 下へ移動   |
| Z          | 左回転     |
| X          | 右回転     |

## 対応デバイス

- PC
